# Hello CI/CD

Project demo sederhana CI/CD menggunakan GitHub Actions.

## Deploy (GitHub Pages)

Setelah repo kamu di-push dan GitHub Pages diaktifkan, website akan live di:

- `https://mulyaibrahimaksan.github.io/dem/`

## Cara Menjalankan

```bash
npm install
npm test
```

## Struktur Project

- app.js
- test/app.test.js
- .github/workflows/ci.yml
- index.html
- site.js
